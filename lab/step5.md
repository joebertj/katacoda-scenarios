- awk
- sed
- grep

## awk command

A remarkable pattern scanning and processing language.

## sed command

A powerful stream editor for filtering and transforming text.

`man sed -y info`{{execute}} Type q to exit.

## grep command

`man grep -y info`{{execute}} Type q to exit.
